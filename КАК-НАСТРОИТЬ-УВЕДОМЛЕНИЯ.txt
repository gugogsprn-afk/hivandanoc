  КАК НАСТРОИТЬ ПОЧТУ И TELEGRAM ДЛЯ ЗАЯВОК С САЙТА
═══════════════════════════════════════════════════════════

Сейчас на сайте 3 формы отправляют уведомления:
  • Запись на приём      (appointment.html)
  • Обратная связь       (contacts.html)
  • История пациента     (submit-story.html)

ВАЖНО: обычный СМОТРЕТЬ.bat только показывает сайт.
       Письма и Telegram работают только через:
       → СМОТРЕТЬ-С-УВЕДОМЛЕНИЯМИ.bat


─── ШАГ 1. Установите Node.js (если ещё нет) ───

  Скачайте: https://nodejs.org  (версия LTS)
  Установите с настройками по умолчанию.


─── ШАГ 2. Создайте файл настроек .env ───

  В папке проекта откройте командную строку и выполните:

    copy .env.example .env

  Откройте .env блокнотом и заполните свои данные.


─── ШАГ 3. Настройте ПОЧТУ (тест) ───

  Пример для Gmail:

  1) NOTIFY_EMAIL — ваш email, КУДА приходят заявки
     NOTIFY_EMAIL=ваш@gmail.com

  2) SMTP_USER — тот же Gmail
     SMTP_USER=ваш@gmail.com

  3) SMTP_PASS — НЕ обычный пароль, а «Пароль приложения»:
     • Google аккаунт → Безопасность
     • Двухэтапная аутентификация (включить)
     • Пароли приложений → создать → «Почта»
     • Скопировать 16 символов в SMTP_PASS

  4) Остальное для Gmail оставьте:
     SMTP_HOST=smtp.gmail.com
     SMTP_PORT=587
     SMTP_SECURE=false


─── ШАГ 4. Создайте TELEGRAM БОТА ───

  1) В Telegram найдите @BotFather
  2) Отправьте:  /newbot
  3) Придумайте имя и username бота
  4) BotFather даст TOKEN — вставьте в .env:
     TELEGRAM_BOT_TOKEN=123456789:ABC...

  5) Узнайте CHAT_ID (куда слать уведомления):
     • Напишите своему боту любое сообщение (например: привет)
     • В браузере откройте (подставьте свой TOKEN):
       https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
     • Найдите:  "chat":{"id": 123456789
     • Это число вставьте в .env:
       TELEGRAM_CHAT_ID=123456789


─── ШАГ 5. Запуск и проверка ───

  1) Запустите:  СМОТРЕТЬ-С-УВЕДОМЛЕНИЯМИ.bat
  2) В окне должно быть:
       Почта: включена
       Telegram: включён
  3) Откройте сайт → Контакты → отправьте тестовое сообщение
  4) Проверьте:
       • почту NOTIFY_EMAIL
       • Telegram — сообщение от бота


─── Проверка без формы ───

  Откройте в браузере:
    http://127.0.0.1:8765/api/health

  Должно показать:
    {"ok":true,"email":true,"telegram":true}


─── Если не работает ───

  • Почта не приходит — проверьте пароль приложения Gmail
  • Telegram молчит — написали ли боту первым? CHAT_ID верный?
  • Ошибка в окне сервера — читайте текст ошибки
  • .env не читается — файл должен быть в папке проекта, не .env.example

Файл .env в git не попадает (секреты в безопасности).
